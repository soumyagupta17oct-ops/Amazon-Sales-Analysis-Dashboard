# Amazon-Sales-Analysis-Dashboard
Amazon Sales Dashboard
dynamic, interactive data visualization tool built to explore amazon product sales performance.
This dashboard helps track key metrics, analyze trends, and uncover top performing products using intuitive visuals and filters .It's designed for business users seeking actionable insights from sales data.

The dashboard was built using using the following tools and technologies.
*Power Bi : Main data visualization platform used for report creation.
*Power Query: Data transformation and cleaning layer for reshaping and preparing the data.
*DAX(Data Analysis Expression): used for calculated measures,dynamic visuals and conditional logic.
Data Modelling & Transformation: structured disparate sales data for efficient querying and reporting.
Goal & Business Problems:This dashboard addresses the need for a clear,visual summary of Amazon sales performance .it helps stakeholders identify seasonal trends ,top-selling products, and review patterns to guide inventory planning, marketing strategies and customer engagement efforts.

Dashboard Highlights
KPIS:YTD Sales,QTD Sales, YTD Product Sold,YTD Reviews
Charts:
* Sales by Month
* Sales by Week
* Sales by Product Category
* Top 5 Products by YTD Sales
* Top 5 Products by YTD Reviews
  Slicers : Product Selector,Quarter Filter

Design:Clean layout with custom color palette and responsive visuals.
